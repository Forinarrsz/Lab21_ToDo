## Описание
В ходе данной работы было изучено создание динамического веб-приложения "Список задач" (TODO-лист). Основной упор был сделан на:
* **Манипуляцию DOM:** поиск элементов, создание новых узлов (`createElement`) и изменение структуры дерева (`appendChild`, `remove`).
* **Обработку событий:** использование `addEventListener` для отслеживания кликов и нажатий клавиш (Enter).
* **Работу с формами:** получение и очистка данных из полей ввода.
* **Стилизацию:** создание адаптивного и интерактивного интерфейса с помощью CSS.

## Структура проекта
* `index.html` главная страница.
* `todo.html` страница приложения TODO.
* `style.css` стили для `index.html`
* `todo.css`  стили приложения.
* `todo.js`  логика работы на JavaScript.
* `main.js` script для `index.html`
* `img/`  папка со скриншотами выполненной работы.
